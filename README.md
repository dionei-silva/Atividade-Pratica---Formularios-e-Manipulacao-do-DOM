## Desafios propostos!!
- Desenvolva uma página HTML que replique a imagem acima, contendo um formulário com todos os itens mostrados.
- Todos os campos de dados pessoais devem ser preenchidos obrigatoriamente pelo usuário.
- Ao carregar a página deve vir previamente marcado um dos radios buttons.
-  O usuário deve marcar ao menos um checkbox. Deve ser validado o preenchimento desse input.
-  O campo de texto de label “mini-curriculo” não é obrigatório o preenchimento pelo usuário.
-  Capture os dados preenchidos no formulário utilizando os métodos de manipulação do DOM. Ao final, mostre no console.log os dados preenchidos pelo usuário.

- A resolução dos desafios estão no console.log
